# Derin-Ogrenme-ile-Saglikli-ve-Sagliksiz-Retina-Optik-Disk-Goruntulerinin-Taninmasi
"https://www.kaggle.com/datasets/andrewmvd/retinal-disease-classification/data" linkteki veri setiyle retina optik disk görüntülerini inceleyerek optik diskin sağlıklı veya sağlıksız olup olmadığını tanımayı amaçlayan bir proje planlanmaktadır. Çalışmada iki farklı derin öğrenme modeli kullanılacaktır: Sequential model ve U-Net modeli. Sequential model ile sınıflandırma performansının, U-Net modeli ile ise optik disk bölgesinin segmentasyon performansının değerlendirilmesi planlanmaktadır. Modellerden sırasıyla yaklaşık %90 ve %80 doğruluk seviyelerine ulaşılması hedeflenmektedir. Şimdilik daha basic model olan sequential cnn ile başlıyoruz ve ileriki zamanlarda u-net gibi karışık bir modeli de denemeyi amaçlamaktayız.
